# 📄 RPN — Estimation des coûts du projet

## 📌 Présentation

Le projet **RPN** est une plateforme communautaire web développée en **PHP/MySQL** avec une architecture **MVC personnalisée**.

Il inclut :

- 🔐 Connexion via Google OAuth
- 👤 Espace membre avec dashboard
- 🛠️ Espace administrateur complet
- 🧠 Architecture MVC maison
- 🛡️ Sécurité anti-bruteforce (blocage IP)
- 👑 Gestion des rôles (admin / super-admin)
- 🎨 Système de thèmes dynamiques
- ⚙️ Configuration dynamique (JSON)
- 🗄️ Auto-création de base de données

---

## ⚙️ Décomposition des coûts
