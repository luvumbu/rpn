Manipule Pythagore avec tes doigts

Ici, rien à coder : attrape les points et déplace-les. Les longueurs, les carrés et l'égalité a² + b² = c² se recalculent en temps réel.

Comment faire ? Clique (ou touche) un gros point coloré et fais-le glisser. Les nombres à droite suivent le mouvement.

1 Le triangle vivant

Déplace le sommet B (en bas) et le sommet C (en haut). L'angle droit reste en A. Regarde les trois carrés grandir et l'égalité se vérifier toute seule.

a (vertical) = 0
b (horizontal) = 0
c (hypoténuse) = 0
a² + b² = 0
c² = 0
a² + b² = c² ✓
Les carrés des côtés sont dessinés à l'échelle : l'orange (sur c) a toujours l'aire des deux autres réunis.

2 Est-il rectangle ?

Ce triangle-là n'a pas forcément d'angle droit. Déplace librement le sommet S : l'outil teste si le plus grand côté au carré égale la somme des deux autres au carré. Trouve une position où il devient rectangle !

côtés : –
grand² = 0
somme des 2 autres² = 0
Pas rectangle
L'angle le plus proche de 90° est surligné. Quand l'écart est minuscule, le triangle est (presque) rectangle.

3 La distance entre deux points

Déplace P et Q. Le trajet horizontal et vertical forme un triangle rectangle, et la ligne directe (l'hypoténuse) est la distance — exactement le calcul du GPS.

Δ horizontal = 0
Δ vertical = 0
distance PQ = 0
distance = √(Δhoriz² + Δvert²)
